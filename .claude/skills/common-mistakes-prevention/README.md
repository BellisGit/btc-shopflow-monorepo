# 常见错误预防 Skill

本 skill 记录项目中反复出现的常见错误及其正确的解决方案。

## 文件结构

```
common-mistakes-prevention/
├── SKILL.md          # 主要技能文档
└── README.md         # 本文件
```

## 记录的常见错误

1. **国际化配置位置错误** - 应该在 config.ts 中配置
2. **Label 没有 name 的情况** - 使用 `_hideLabel: true`
3. **组件选择错误** - 优先使用 `btc-*` 自定义组件
4. **不必要的布局美化** - 避免添加背景色、标题、卡片等
5. **Padding 间距不规范** - 统一使用 10px

## 使用方法

当 AI 助手编写代码时，应该：

1. 自动检查本 skill 中记录的常见错误
2. 遵循正确的做法
3. 在代码审查时使用检查清单

## 与其他 Skills 的关系

- `page-creation-guide` - 页面创建指南，会引用本 skill 的规范
- 在创建新页面时，应该同时参考这两个 skills

## 更新原则

当发现新的常见错误时，应该：

1. 添加到本 skill 中
2. 更新检查清单
3. 在相关 skills 中引用
