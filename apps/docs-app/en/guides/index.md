# Development Guides

This documentation library provides complete development guides, including frontend component development, backend service development, system configuration, and deployment guides, helping you quickly get started and deeply understand the BTC Workshop Management System.

## Core Features

- **Out of the Box**: Provides complete CRUD components, simplifying business development
- **Type Safety**: Based on TypeScript, provides complete type definitions
- **Theme Customization**: Supports dynamic theme switching and custom styles
- **Internationalization**: Built-in multi-language support
- **Responsive**: Adapts to various screen sizes

## Quick Start

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev
```

## Design Philosophy

We are committed to providing:
- **Simple and Easy to Use**: Lower learning curve, improve development efficiency
- **Flexible and Extensible**: Supports custom configuration and extensions
- **Performance Optimization**: Focus on load speed and runtime performance
- **Development Experience**: Provides complete development tools and debugging support
