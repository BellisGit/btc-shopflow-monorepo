# Document Templates

This provides standard templates for various document types to maintain document format consistency.

## Template List

### Architecture Decisions
- **[ADR Template](./adr-template)** - Architecture Decision Record template

### Component Documentation
- **[Component Template](./component-template)** - Component documentation template

### Solution Design
- **[RFC Template](./rfc-template)** - Solution design document template

### Operating Procedures
- **[SOP Template](./sop-template)** - Standard Operating Procedure template

## Usage Guide

1. **Select Appropriate Template**: Choose corresponding template based on document type
2. **Copy Template Content**: Copy template content to new document
3. **Fill Specific Information**: Replace placeholders in template
4. **Maintain Format Consistency**: Ensure following template format requirements

## Template Standards

- All templates include standard frontmatter
- Use unified heading hierarchy
- Maintain consistent format style
- Include necessary metadata
