# 时间线

> 最近的文档更新

::: tip 提示
目前还没有收录任何文档请运行 `pnpm --filter docs-site ingest` 来收录文档
:::

## 如何开始

### 1. 创建新文档

```bash
pnpm --filter docs-site new-doc
```

### 2. 为现有文档添加 frontmatter

```bash
pnpm --filter docs-site add-frontmatter
```

### 3. 收录文档

```bash
pnpm --filter docs-site ingest
```

收录完成后，时间线会按时间顺序显示最近 90 天的文档更新
